Docker命令

命令 说明 参数 参数说明
docker search <镜像名> 搜索镜像
docker pull [ip地址][:端口号/][镜像名][:版本号] 拉取镜像 ip地址:端口号/ 不填，默认是官方仓库地址
docker images 查看镜像列表
docker tag <镜像>
docker save <镜像> -o <路径/镜像快照名> 导出镜像到本地
docker load -i <路径/镜像快照名> 加载本地镜像文件
docker image prune 删除虚悬镜像
docker image rmi <镜像> 删除镜像
docker image rm <容器> 删除容器 -f 删除运行中的容器
docker run [-it] [--rm] [镜像名][:版本号] [镜像的命令] 启动一个新容器 -it -i交互式操作，-t是终端
--rm 容器退出后，将其删除
镜像名:版本号 也可以用容器名
bash 交互式
docker ps [-a] 查看容器 -a 查看所有的容器；没有-a只查看运行中的容器
docker exec -it <容器> bash 以交互方式进入容器
docker start -i <容器> 启动容器 -i 控制台启动容器；没有-i后台启动
docker rename 重命名容器
docker export <容器> > <路径/容器快照名> 导出容器到本地
cat <路径/容器快照名> \ docker import - <镜像名>/ 本地快照导入为镜像
docker commit <容器> <镜像名>/ 使用容器创建镜像
docker cp <本地文件地址> <容器>:<容器内路径> 拷贝本地文件到容器
docker cp <容器>:<容器内路径> <本地文件地址> 拷贝容器内文件到本地
